# 📉 Mid-Sprint Scope Adjustment Note

**Date:** April 28, 2026  
**Feature:** Nested Threaded Comments  
**Decision:** Cap nesting at Level 1 ✅

## 🚩 Problem Statement
During implementation, the **Full Stack Developer** identified that the current Supabase RLS and recursive frontend components were causing performance lag and logic errors when nesting exceeded two levels.

## 🛠 Adjusted Requirement
- Users can reply to a Top-Level Post.
- Users can reply to a Comment (Level 1).
- **Replies to Replies (Level 2+) are disabled for this iteration.**

## 📊 Impact on Metrics
- **Story Points Removed:** 5 pts.
- **Estimated Dev Time Saved:** 12 hours.
