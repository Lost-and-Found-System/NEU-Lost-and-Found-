# 🧠 Lessons Learned & Recommendations

### 1. API Reliability
* **Lesson:** Gemini's rate limits were a significant bottleneck. 
* **Recommendation:** Always implement a fallback AI provider (as we did with Claude) during the initial architecture phase.

### 2. Database Security (Supabase)
* **Lesson:** Row Level Security (RLS) is powerful but adds complexity to debugging. 
* **Recommendation:** Dedicate more time in Sprint 1 to defining RLS policies to avoid "Visibility Meltdowns" later.

### 3. Image Management
* **Lesson:** Cloudinary significantly reduced frontend complexity compared to manual image handling. ✅

### 4. Scope Management
* **Lesson:** Capping nested replies at 1 level was the right move for the timeline.
