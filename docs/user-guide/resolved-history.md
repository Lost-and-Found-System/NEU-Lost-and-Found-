# Archive Hall - Resolved History

The **Resolved Tab** allows users to browse the community's successful returns and resolutions. All records here are **archived and preview-only**—no edits or deletions are permitted.

## Access

Click the **Resolved** tab.

## Stats Overview

At the top of the page, you'll see the total number of resolutions displayed (e.g., "13 RESOLUTIONS").

## Search & Filter Options

The page provides the following controls in the upper section:

| Control | Description |
|---------|-------------|
| **Search bar** | Search resolved items by title, description, or ID (placeholder: "Search resolved items...") |
| **Date range picker** | Two date fields (dd/mm/yyyy) to filter by resolution date range |
| **Category filters** | Filter by item type: All, Electronics, Clothing, ID/Cards, Keys, Jewelry, Others |

## Resolution Cards (Grid View)

Resolved items are displayed as **cards** in a responsive grid layout. Each card shows:

- **Status badge:** RESOLVED + category badge (LOST or FOUND)
- **Item name/title** (e.g., "Teddybear keychain")
- **Description** (truncated, e.g., "brown teddy bear keychain")
- **Location/ID** (e.g., "M101", "Room M105")
- **Resolution date** (e.g., "May 10, 2026, 08:00AM")
- **Owner/Reporter name** (e.g., "ESPINOSA, MARIA")
- **ARCHIVED badge** (confirms read-only status)

> ⚠️ **Note:** Anonymous reports show as "ANONYMOUS" instead of a name.

## Read-Only Archive Behavior

- ❌ **No edits** – Archived resolutions cannot be modified
- ❌ **No deletions** – Records are permanent for audit/history purposes
- ✅ **Full visibility** – All details remain searchable and viewable
- 🔒 **ARCHIVED badge** – Confirms the item is locked

## Pagination

If more resolutions exist than fit on one page, pagination controls appear at the bottom of the grid.
