# Notifications System

## Notification Tab & Count
- ✅ Number of unread notifications appears beside "Notification" tab
- ✅ Count updates when notifications are read
- ✅ Count disappears when no unread notifications remain

## Notification Page
- ✅ Clicking "Notification" tab navigates to the Notifications page
- ✅ Page title shows "Notifications"
- ✅ "Clear all" button is present
- ✅ "Stay updated with your activities" message is displayed

## Notification List
Each notification shows:
  - ✅ Name of who performed the action
  - ✅ Action description (commented, replied)
  - ✅ Target item name in quotes
  - ✅ Timestamp (e.g., MAY11,2026,07:14PM)

## Clear All Functionality
- ✅ Clicking "Clear all" removes all notifications
- ✅ After clearing, page shows empty state

## Real-time Behavior
- ✅ New notifications appear without page refresh
- ✅ Count updates in real time

## Edge Cases
- ✅ Empty state shows appropriate message when no notifications exist
- ✅ Handles slow network gracefully
