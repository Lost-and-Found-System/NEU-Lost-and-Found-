# Test Case TC-009

| Field | Value |
|-------|-------|
| **Test ID** | TC-009 |
| **Feature** | Edge case – Empty inputs |
| **Scenario** | User attempts to post an item with empty required fields |
| **Steps** | 1. Click "Post Item" <br> 2. Leave title empty <br> 3. Fill description <br> 4. Select lost/found and category <br> 5. Click "Post Listing" |
| **Expected Result** | Error message: "Please fill out this field." |
| **Actual Result** | Post cannot upload because the title is empty. Browser highlighted the empty title field with validation message: "Please fill out this field." Post was not created. |
| **Status** | Pass |
| **Notes** | Validation prevents submission when title is empty. User must enter a title before posting. No server request made. |
