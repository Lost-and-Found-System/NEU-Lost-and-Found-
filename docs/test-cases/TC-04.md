# Test Case TC-004

| Field | Value |
|-------|-------|
| **Test ID** | TC-004 |
| **Feature** | View all resolved issues by user |
| **Scenario** | User views their own resolved issues after resolving multiple posts |
| **Steps** | 1. User resolves 2 different posts (one lost, one found) <br> 2. Navigate to "My Resolved Issues" page<br> 3. Confirm the posts are in the "Resolved" tab. |
| **Expected Result** | Both resolved posts appear with correct details |
| **Actual Result** | Navigated to "Resolved" tab. Both resolved posts (one lost, one found) displayed in the "Resolved" tab. Each post showed title, original category (lost/found), and original description. |
| **Status** | Pass  |
| **Notes** | Resolved tab only shows posts marked as resolved by the logged-in user. Unresolved posts do not appear in this tab. Tested with 2 posts – both appeared correctly. |
