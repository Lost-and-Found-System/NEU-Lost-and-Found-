# Test Case TC-010

| Field | Value |
|-------|-------|
| **Test ID** | TC-010 |
| **Feature** | Negative test – Unauthorized resolution |
| **Scenario** | Non-owner tries to resolve someone else's post |
| **Steps** | 1. User A creates a lost item post <br> 2. User B logs in <br> 3. User B navigates to User A's post <br> 4. User B clicks "Resolve Issue" button |
| **Expected Result** | Action fails; button hidden/disabled for non-owners |
| **Actual Result** | Resolve button is only visible on the user's posts. User B could not see any resolve button on User A's post. |
| **Status** | Pass |
| **Notes** | Resolve button conditionally rendered based on post ownership. Non-owners cannot resolve others' posts. Confirmed with PM – this is intended behavior. |
