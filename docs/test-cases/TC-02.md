| Field | Value |
|-------|-------|
| **Test ID** | TC-002 |
| **Feature** | Commenting on a post + Notifications |
| **Scenario** | User comments on another user's post; owner receives notification |
| **Steps** | 1. User A logs in and creates a post <br> 2. User B logs in <br> 3. User B navigates to User A's post <br> 4. User B adds comment: "I think I saw this at room M111" <br>5. Click "+" button to add comment <br>6. User A checks notifications |
| **Expected Result** | Comment appears under post; User A receives a notification: "User B commented on your post" |
| **Actual Result** | Comment "I think I saw this at room M111" displayed under User A's post with User B's name and timestamp. User A's notification bell showed red badge (1).
