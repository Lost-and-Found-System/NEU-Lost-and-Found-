# Test Case TC-006

| Field | Value |
|-------|-------|
| **Test ID** | TC-006 |
| **Feature** | Search bar |
| **Scenario** | User searches for a keyword that exists in post titles |
| **Steps** | 1. In search bar, type a keyword you want to search <br> 2. Press Enter |
| **Expected Result** | Returns the exact matching post |
| **Actual Result** | Typed "Teddy bear" in search bar. Returned 1 post. Typed non-existent keyword "xylophone" – returned "No items found matching your criteria." |
| **Status** | Pass |
| **Notes** | Search is case-insensitive – "teddy bear" and "TEDDY BEAR" returned same results. Search applies to post titles only (not descriptions per current spec – confirmed with PM). Search works together with active filters (e.g., search + "Lost" filter shows only lost items matching keyword). Empty search returns to full unfiltered feed. |
