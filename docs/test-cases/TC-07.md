# Test Case TC-007

| Field | Value |
|-------|-------|
| **Test ID** | TC-007 |
| **Feature** | View all my posts |
| **Scenario** | User navigates to view all posts they have created |
| **Steps** | 1. User uploads  different posts (lost/found mixed) <br> 2. Click "My Posts" in navigation menu |
| **Expected Result** | All posts appear; each shows title, category, lost/found status, and resolution status |
| **Actual Result** | Navigated to "My Posts" page. All posts displayed correctly: 1 lost item (others) and 1 found item (electronics). Each post card showed title, category, lost/found badge, resolution status (unresolved/resolved), and date posted. Posts ordered by most recent first. |
| **Status** | Pass |
| **Notes** | "My Posts" only shows posts created by the logged-in user. No other users' posts appear. Resolution status is clearly marked with "Resolved" badge on resolved posts. |
